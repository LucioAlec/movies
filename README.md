# Movies

Command-line app/gem to play and review movies.

##  Features
- Load a movie list from CSV.
- Review system using a virtual die (thumbs up/down/skip).
- Snack bar with carbs tracking per movie.
- 3D movies (`Movie3D`) with a “wow factor” that boosts thumbs ups.
- Save final rankings to a text file.
